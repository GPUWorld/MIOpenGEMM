find¶

Solution MIOpenGEMM::find(float allotted_time, cl_command_queue command_queue, cl_mem a, cl_mem b, cl_mem c, bool enforce_determinism, const Geometry &tgg, bool verbose, bool with_warnings)¶

Find a Solution, considering only kernels without workspace requirements.

Return
The best Solution found during search
Parameters
  • allotted_time: An upper bound in seconds on the time spent searching. When this time
  • command_queue: An OpenCL command queue has expired, the best Solution found thus far is returned.
  • a: Matrix A, memory will be unchanged
  • b: Matrix B, memory will be unchanged
  • c: Matric C, memory will be unchanged
  • enforce_determinism: If true, only kernels which are bitwise consistent are considered. For small GEMM problems enforce_determinism = False finds faster Solutions.
  • tgg: GEMM problem Geometry as in geometry.hpp
  • verbose: Print the search summary to std::cout
  • with_warnings: Print warnings to std::cerr. Warnings are provided when the allotted_time is insufficient to find good Solutions.

Solution MIOpenGEMM::find(cl_command_queue command_queue, const FindParams &find_params, cl_mem a, cl_mem b, cl_mem c, cl_mem workspace, const std::string constraints_string, const Geometry &gg, const Offsets &toff, outputwriting::OutputWriter &mowri, bool c_is_const, bool use_mowri_tracker)¶

Find a Solution, considering kernels with and without workspace requirements. This function is experimental.

Return
best Solution found during search
Parameters
  • command_queue: OpenCL command queue
  • find_params: Parameters define duration and structure of search
  • a: Matrix A
  • b: Matrix B
  • c: Matric C
  • workspace: Workspace memory
  • constraints_string: String describing the constraints on the kernels considered. It is a substring of a hyperstring, defining with hyper-parameters have fixed values
  • gg: GEMM problem Geometry as in geometry.hpp
  • toff: The offsets of a,b,c and workspace
  • mowri: Controls writing to terminal and file
  • c_is_const: Whether matrix C may be modified
  • use_mowri_tracker: Controls writing to terminal of search summary

get_default¶

Solution MIOpenGEMM::get_default(cl_command_queue command_queue, std::string constraints_string, const Geometry &gg, std::string k_comment, outputwriting::OutputWriter &mowri)¶

Return a default Solution from cache, without exploring. If no Solution exists, an error will be thrown.

Return
best Solution found during search
Parameters
  • command_queue: OpenCL command queue, used to extract device name
  • constraints_string: String describing the constraints on the kernels considered. It is a substring of a hyperstring, defining with hyper-parameters have fixed values
  • gg: GEMM problem Geometry
  • k_comment: If a Solution from a custom cache is to be returned, this can be a non-empty string
  • mowri: Controls writing to terminal and file
